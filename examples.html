How to Use DevPulse

Search by Username

Simply enter any GitHub username and press Enter to see their activity dashboard instantly. No authentication required!

Share Profiles

After viewing a profile, copy the URL to share it. Links like devpulse/index.html?user=torvalds will auto-load the profile.

View Your Own Profile

Go back to DevPulse and search for your GitHub username to see your personal activity dashboard.

Features

DevPulse shows you:

  • ✓ Profile information (avatar, name, bio, followers)
  • ✓ 12-month contribution heatmap (like GitHub's graph)
  • ✓ Top programming languages (with a beautiful bar chart)
  • ✓ Top 5 repositories by stars
  • ✓ Commit streaks (current and longest)
  • ✓ Full mobile responsiveness

Technical Details

Zero Backend

DevPulse is a static web app. All data is fetched from GitHub's public API directly in your browser. No server, no backend, just client-side JavaScript.

No Build Step

Open index.html in your browser and it works immediately. No npm install, no build process, no dependencies to manage.

Privacy

Only public GitHub data is fetched. No tracking, no analytics, no cookies. Your data is fetched directly from GitHub's API — we don't store anything.
